# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-XX

### Added
- Complete TypeScript rewrite with strict typing
- Comprehensive test suite with >70% coverage
- Proper error handling and validation
- CLI with improved UX and error messages
- Support for multiple frameworks (Next.js, React, Vue, Angular, etc.)
- Module generation with templates
- Configuration management system
- Migration tools from legacy systems

### Changed
- Renamed from Nextisan to Modularisan
- Improved CLI command structure
- Better file organization and naming conventions

### Fixed
- Various bug fixes from beta version
- Improved Windows compatibility
- Better error messages

## [0.1.0-beta.1] - 2024-XX-XX

### Added
- Initial beta release
- Basic module generation
- Framework detection
